## 83. 배열의 기본 문법

- **배열의 이름**은 0번 요소의 메모리 주소에 대한 식별자이며 **상수**(l-value가 아니다)

  - 문자열은 이름이 없는 문자 배열

  - 그래서 아래는 가능하다.
    ```c
    char* str = "banana";
    ```

    - 이런건 안돼고 ~`int* nbr = 23;`~
