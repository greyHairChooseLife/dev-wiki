## 72. switch-case문


```c
switch (<something>):{
  case <A>:  /* something == A 상등 연산으로 비교한다. */
    ...
    break;   /* 조건 만족 시 탈출을 원하면 직접 해줘야한다. */

  case <B>:  /* something == B 상등 연산으로 비교한다. */
    ...
    break;

  default:   /* 기본값 */
    ...
}
```

- disassemble해서 if/else와 비교해보는것도 좋겠다.
