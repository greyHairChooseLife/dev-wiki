


C 프로그램에서 `main()` 함수가 실행되는 **메인 쓰레드**도 "쓰레드"입니다.  
즉, 프로그램이 시작되면 운영체제가 **메인 쓰레드(=main 함수가 실행되는 쓰레드)**를 하나 만들어서 실행합니다.

### 정리

- `main()` 함수가 실행되는 것은 **메인 쓰레드**의 동작입니다.
- pthread를 사용하지 않아도, 프로그램에는 항상 하나의 쓰레드(메인 쓰레드)가 있습니다.
- pthread를 사용해서 추가로 쓰레드를 만들면, 그때부터 여러 쓰레드가 동시에 실행될 수 있습니다.

### 예시

````c
int main() {
    // 이 코드는 메인 쓰레드에서 실행됨
    return 0;
}
````

여기서도 "쓰레드"라는 개념이 적용됩니다.  
즉, `main()` 함수가 실행되는 것도 쓰레드의 실행입니다.
