# how to learn something?

1. 공부 하려는 주제에 대해 조사한다.
2. 설명을 적어 본다. 가상의 10살 어린이를 앞에 두고.
3. 잘 모르는 부분, 지나치게 복잡한 부분(전문용어를 사용했다던가)을 찾는다.
4. 3.에서 찾은 내용을 더 쉽고 단순하게 설명하기 위해 추가적으로 조사한다.
